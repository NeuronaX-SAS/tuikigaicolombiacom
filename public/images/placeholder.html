Ikigai Template Placeholders

This file is a placeholder to explain where to place the Ikigai template images. The following image files need to be placed in this directory:

  • IKIGAI_Base_Gris.png - Gray Ikigai template
  • IKIGAI_Base_Naranja.png - Orange Ikigai template
  • IKIGAI_Base_Roja.png - Red Ikigai template
  • IKIGAI_Base_Verde.png - Green Ikigai template
  • IKIGAI_EJEMPLO_Azul.png - Blue Ikigai example
  • IKIGAI_EJEMPLO_Morado.png - Purple Ikigai example

These images should be placed in the /public/images/ directory of the project. They will be loaded dynamically by the Ikigai template selector in the user interface.
